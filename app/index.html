Project Overview:

As part of our hands-on learning experience at AKumoSolutions DevOps bootcamp, our team, consisting of 10 students form the April batch and our instructor, have embarked on our first collaborative project: the design, development, and deployment of a web application.

The goal was to apply our newly acquired skills so far in a real-world context and gain practical experience in both front-end and back-end development workflows, in monitoring and keeping the application running and gain experience in working on GitHub platform.

Objectives:

  • Develop a complete web application
  • Collaborate effectively using version control and Github tools
  • Deploy the application.
  • Keep the application running by logging and monitoring
